#######Colors

-------Background Gradients:-------
-Main background: from-slate-50 via-blue-50 to-indigo-100
-Button gradients: from-blue-500 to-indigo-600, from-blue-600 to-indigo-700
-Icon backgrounds: from-blue-500 to-indigo-600

-------Text Colors:--------------
-Primary headings: from-slate-800 via-blue-600 to-indigo-600 (gradient)
-Secondary text: text-slate-600, text-slate-700, text-slate-800
-Muted text: text-slate-500, text-slate-400

-------Interactive Elements:-------
Blue accents: text-blue-600, hover:text-blue-800, border-blue-200
Red delete buttons: text-red-600, border-red-200, hover:bg-red-50
Success toasts: Uses theme colors for success states

-------Card/Surface Colors:-------
Cards: bg-white/80 (80% opacity white with backdrop blur)
Hover states: hover:bg-blue-50/50
Borders: border-slate-100, border-slate-200
Icons (from Lucide React)
BookmarkPlus - Add bookmark button and form header
Bookmark (as BookmarkIcon) - Table header and empty state
Globe - URL fields and links
Tag - Name/label fields
ExternalLink - Visit website buttons
Trash2 - Delete buttons
Calendar - Created date display

#######Fonts
The website uses the default system font stack defined by Tailwind CSS, which includes:

-Primary: ui-sans-serif, system-ui, sans-serif
-Fallbacks: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial
-Font Weights Used:

font-bold (700) - Main headings
font-semibold (600) - Section titles
font-medium (500) - Labels and buttons
font-normal (400) - Body text
